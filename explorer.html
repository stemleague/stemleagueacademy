STEM League Explorer Program

STEM League Explorer Program is a 5 week school-year opportunity for aspiring 6th - 12th grade students to learn coding through live instruction from STEM League Instructors. As an Explorer, students will work together to learn coding and create large scale projects while building their technical skills in a tight-knit student community.





Explorer Program Dates:

Winter Cohort: January 31 - March 4, 2022 (Application Deadline: January 25)
Spring Cohort: March 14 - April 15, 2022 (Application Deadline: March 9)

What does a typical week look like?

  • One 60 minute lesson via Zoom
  • Office Hours via Discord throughout the week
  • One Coding Practice Assignment
  • All lessons and office hours times will be announced once students are admitted.

Applicant Qualifications:

  • Enthusiastic learner with a curiousity to learn coding (no experience required)
  • Must be a rising 6th-12th grade student

At the end of the Program, students will:

  • Build strong technical coding skills
  • Develop a technical portfolio of coding projects for future intern/job opportunities
  • Stay connected with STEM League Community for future opportunities

SEE TRACKS





Computer Programming I

This class dives into the fundamentals of coding with Java where students will learn how to write scalable and efficent programs.

Topics Covered: Basic Java Data Types, Method Decomposition, Conditionals, Parameters, Code Style, Writing scalable Java Programs for algorithms


Web Development

In this class, students will embark on learning how the web works and how to build their own portfolio website in HTML/CSS and Javascript.

Topics Covered: HTML Basics, CSS Layout, Accessibility, UX Design, Javascript Basics


Data Programming

This class is for students new to coding where we will learn how to build complex programs and perform data analysis with Python.

Topics Covered: Python Data Types, Basic Data Structures, Data Science (Graphing, Analysis), Interactive User Programs with Python
